ວິທີປ່ຽນລະຫັດຜ່ານ

  1. ຫຼັງ Login ສຳເລັດ → ຄລິກ tab ⚙️ ຕັ້ງຄ່າ
  2. ເລື່ອນລົງຫາສ່ວນ "🔑 ປ່ຽນລະຫັດຜ່ານ"
  3. ໃສ່ ລະຫັດໃໝ່ ແລ້ວໃສ່ ຢືນຢັນ ອີກຄັ້ງ
  4. ຄລິກ "ປ່ຽນ" — ຈະມີຂໍ້ຄວາມຢືນຢັນ
💡 ລະຫັດຜ່ານຖືກເກັບໄວ້ໃນ Browser (localStorage) ຂອງຄອມພ໌ທ່ານ. ຫາກເປີດ Browser ໃໝ່ ຫຼື ຜ່ານ Private Mode ອາດຕ້ອງໃສ່ລະຫັດໃໝ່.

  1. ຄລິກ "💾 ບັນທຶກ" — ຜະລິດຕະພັນຈະສະແດງໃນໜ້າລູກຄ້າທັນທີ

ຜະລິດຕະພັນ VIP

ຜະລິດຕະພັນ VIP ຈະ ບໍ່ສະແດງ ໃຫ້ລູກຄ້າທົ່ວໄປ — ເຫັນສະເພາະລູກຄ້າ VIP ທີ່ກຳໜົດໄວ້ເທົ່ານັ້ນ.

  1. ໃນຟອມຜະລິດຕະພັນ → ໝາຍ tick "👑 ຜະລິດຕະພັນ VIP"
  2. ຊ່ອງ "ເບີໂທລູກຄ້າ VIP" ຈະເປີດຂຶ້ນ — ໃສ່ເບີໂທລູກຄ້າ VIP ຄັ່ນດ້ວຍ ,
  3. ຕົວຢ່າງ: 020xxxxxxxx, 030xxxxxxxx
  4. ຄລິກ "💾 ບັນທຶກ"
💡 ລູກຄ້າ VIP ຈະເຫັນ badge 👑 VIP Exclusive ໃນໜ້າເລືອກຜະລິດຕະພັນ ຫຼັງຈາກໃສ່ເບີໂທຂອງຕົນ.
🔁

ການຕັ້ງດອກທົບຕົ້ນ (Compound)

ດອກທົບຕົ້ນໝາຍຄວາມວ່າ: ຖ້າລູກຄ້າ ບໍ່ຊຳລະທັນກຳໜົດ, ຍອດດອກຄ້າງຈະຖືກລວມໃສ່ຕົ້ນ ແລ້ວຄິດດອກໃໝ່.

  1. ໃນຟອມຜະລິດຕະພັນ → ໝາຍ tick "ດອກທົບຕົ້ນ (Compound)"
  2. ໃສ່ "ທົບຕົ້ນຫຼັງຈາກ (ວັນ)" — ຈຳນວນວັນຫຼັງຄົບກຳໜົດທີ່ຈະທົບ
ຄ່າຄວາມໝາຍ
0ທົບຕົ້ນທັນທີເມື່ອຄົບກຳໜົດ
7ທົບຕົ້ນຫຼັງຈາກ 7 ວັນ ຫຼັງຄົບກຳໜົດ
⚠️ ລະບົບຈະສ້າງ "ສັນຍາກູ້ໃໝ່" ໂດຍໃຊ້ (ຕົ້ນ + ດອກຄ້າງ) ເປັນຕົ້ນໃໝ່. ລູກຄ້າຈະເຫັນຄຳເຕືອນໃນໜ້າຕິດຕາມສະຖານະ.
⚡

✅/⏸ ເປີດ/ປິດສະລັບສະຖານະ active/inactive — ຜະລິດຕະພັນທີ່ inactive ຈະບໍ່ສະແດງໃຫ້ລູກຄ້າ

📥 ການອະນຸມັດລາຍການກູ້

👁️

ເບິ່ງຄຳຮ້ອງ

  1. ຄລິກ tab 📥 ຄຳຮ້ອງ — ຈໍານວນລາຍການລໍຖ້າຈະສະແດງໃນ badge ສີສົ້ມ
  2. ຕາຕະລາງສະແດງ: ວັນທີ, ຊື່ລູກຄ້າ, ເບີໂທ, ຜະລິດຕະພັນ, ຈຳນວນ, ຍອດລວມ
  3. ຄລິກ 👁 ເບິ່ງ ເພື່ອເປີດ Modal ລາຍລະອຽດ — ລວມທັງຮູບຖ່າຍເອກະສານ
💡 ໃນ Modal ລາຍລະອຽດ ທ່ານສາມາດຂະຫຍາຍ/ດາວໂຫລດຮູບຖ່າຍເອກະສານ (ບັດ, ສຳມະໂນ, ພາດສະປອດ) ຂອງລູກຄ້າໄດ້.
✅

ອະນຸມັດ / ປະຕິເສດ

ປຸ່ມການກະທຳສະຖານະໃໝ່
✅ ອະນຸມັດ ຮັບຄຳຮ້ອງ — ລາຍການຍ້າຍໄປ Tab "ລາຍການກູ້" ສະຖານະ "ດຳເນີນ" 💰 ດຳເນີນ
❌ ປະຕິເສດ ຖືກລົ້ມ — ລາຍການຍ້າຍໄປ Tab "ລາຍການກູ້" ສະຖານະ "ປະຕິເສດ" ❌ ປະຕິເສດ
✅ ຫຼັງອະນຸມັດ, ລາຍການຈະປາກົດໃນ Tab 💰 ລາຍການກູ້ ທັນທີ ແລ້ວສາມາດບັນທຶກການຊຳລະໄດ້.
💳

ບັນທຶກການຊຳລະ

  1. ໄປ tab 💰 ລາຍການກູ້ → ຄລິກ 👁 ເບິ່ງ ໃນລາຍການທີ່ຕ້ອງການ
  2. ໃນ Modal → ສ່ວນ "💳 ບັນທຶກຊຳລະ" ຕື່ມ:
ຊ່ອງຄຳອະທິບາຍ
ຈຳນວນຊຳລະຍອດທີ່ລູກຄ້າຊຳລະມາ (ກີບ)
ວິທີຊຳລະເງິນສົດ / BCEL / LDB / ໂອນ
ວັນທີວັນທີຊຳລະຕົວຈິງ
ໝາຍເຫດທາງເລືອກ
  1. ຄລິກ "✅ ບັນທຶກ" — ຍອດໃນລາຍການຈະອັບເດດທັນທີ
  2. ຖ້າຍອດຊຳລະຄົບ → ລາຍການຈະປ່ຽນສະຖານະເປັນ 🎉 ສຳເລັດ
💰 ລາຍການກູ້
ຄົ້ນຫາ, ເບິ່ງລາຍລະອຽດ ແລ້ວຈັດການລາຍການກູ້ທັງໝົດ

ຢືມເພີ່ມ (Top-up)

Top-up ໃຊ້ສຳລັບລູກຄ້າທີ່ຕ້ອງການ ກູ້ເພີ່ມ ໃນຂະນະທີ່ຍັງມີລາຍການດຳເນີນຢູ່.

  1. ເປີດ Modal ລາຍລະອຽດຂອງລາຍການ Active
  2. ຊອກຫາ section "📤 ຢືມເພີ່ມ (Top-up)"
  3. ໃສ່ ຈຳນວນເພີ່ມ ທີ່ລູກຄ້າຕ້ອງການ
  4. ຢືນຢັນ → ຕົ້ນໃໝ່ = ຕົ້ນເກົ່າ + ຈຳນວນເພີ່ມ; ດອກຄຳນວນໃໝ່
⚠️ Top-up ຈະ ຮີເຊັດ ໄລຍະ ແລ້ວ ດອກຄຳນວນໃໝ່ຈາກຕົ້ນລວມ. ກວດໃຫ້ຖ່ວງຖ່ຽນກ່ອນຢືນຢັນ.

  1. ເປີດ Modal ລາຍລະອຽດ → ຊອກຫາປຸ່ມ "🔁 ທົບຕົ້ນ"
  2. ລະບົບຈະສ້າງ "ສັນຍາໃໝ່": ຕົ້ນໃໝ່ = ຕົ້ນ + ດອກຄ້າງ
  3. ປະຫວັດທົບຕົ້ນຈະສະແດງໃນ Modal ແລ້ວ ສະຖານະປ່ຽນເປັນ 🔄 ທົບຕົ້ນ
💡 ທ່ານ (ແລ້ວ ລູກຄ້າ) ສາມາດເຫັນ "ປະຫວັດທົບຕົ້ນ" ໃນ Modal ວ່າທົບຕົ້ນກີ່ຄັ້ງ, ແຕ່ລ່ວັນທີ, ດອກຄ້າງ ແລ້ວ ຕົ້ນໃໝ່.
